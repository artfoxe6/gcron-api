# gcron-api

gcron-api 是 [gcron](https://github.com/artfoxe6/gcron "gcron") 项目的配套管理系统<br />

基于 [quick_gin](https://github.com/artfoxe6/quick_gin "quick_gin") 开发

#### 运行条件：
+ Golang >= 1.12
+ Go Module
+ MySQL/MariaDB

#### 安装运行
git clone https://github.com/artfoxe6/gcron-api.git <br />
cd gcron-api <br />
go run main.go <br />
